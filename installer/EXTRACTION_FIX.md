# Исправление ошибки распаковки

## Проблема

После успешного скачивания архива файл `Tf2SkinGenerator.exe` не найден после распаковки.

## Причина

PowerShell команда `Copy-Item` с рекурсией может работать неправильно при копировании структуры папок. Улучшена логика распаковки.

## Исправление

Использована более надежная команда PowerShell для копирования файлов с сохранением структуры:

```powershell
Get-ChildItem -Path $sourceDir -Recurse | Copy-Item -Destination { Join-Path $destDir $_.FullName.Substring($sourceDir.Length + 1) } -Force
```

Эта команда:
1. Получает все файлы рекурсивно из исходной папки
2. Копирует каждый файл, вычисляя правильный путь назначения
3. Сохраняет структуру папок

## Что нужно сделать

Пересобрать установщик с исправленным файлом `Tf2SkinGenerator-APP-TAG.iss`.

## Альтернативное решение

Если проблема сохраняется, можно проверить структуру архива:
1. Распаковать `dist\Tf2SkinGenerator-windows.zip` вручную
2. Проверить, что внутри есть папка `Tf2SkinGenerator\` с файлом `Tf2SkinGenerator.exe`
3. Убедиться, что файл называется именно `Tf2SkinGenerator.exe` (не `TF2SkinGenerator.exe`)
